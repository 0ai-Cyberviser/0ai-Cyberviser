`0ai-Cyberviser` is the primary GitHub profile for Johnny Watters and the public company identity for `0AI` / CyberViser.

The mission is to build a connected, free-first, GitHub-native cybersecurity engineering ecosystem around AI agents, fuzzing, dataset generation, SOC/IR workflows, and repository maintenance.

## Current focus

- Connect every public repo back to `https://cyberviserai.com/`
- Polish READMEs, badges, links, roadmaps, and GitHub Pages surfaces
- Use GitHub Actions and GitHub Pages before paid infrastructure
- Keep private Drive/Admin/secrets/datasets out of public repos
- Build Hancock, PeachFuzz, PeachTree, and MrClean as one coordinated security toolchain
